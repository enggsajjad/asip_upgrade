FPGA Boards

Virtex 5 - LX Board

This board is the same as the Xilinx ML505 Evaluation Platform, except the FPGA chip is a Virtex 5-LX 110T device. Use the same documentation as the ML505 (which has a Virtex 5-LX 50T chip)

Virtex 5 - FXT Board

This board is very similar to the XUPV5-LX110T board (above). The major change is that it has a different version of the Virtex-5 FPGA. The -FX version is targeted at embedded development

The FX device has 2 PowerPC 440 processors on silicon. It also has some very nice direct-to-fabric I/O options, so your hardware doesn't need to be hung on a processor bus. This results in very low latency operations.
